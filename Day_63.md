    Comparison_Operators
        1. Relational: <, >, <=, >= (Works like C++/Bash).
        2. Loose (==): Coerces types. Avoid this in backend code.
        3. Strict (===): Checks Type + Value. Use this 99% of the time.
        4. Objects: Compared by reference (memory address), not content.
        5. Tip: null === undefined is false; null == undefined is true.

    Logical_Operators
        1. && (AND): Returns first Falsy or last Truthy. (Short-circuits)
        2. || (OR):  Returns first Truthy or last Falsy. (Great for defaults)
        3. !  (NOT): Flips the boolean value.
        4. !! (Double NOT): Forces a value into a Boolean type.
        5. ?? (Nullish): Like ||, but only for null/undefined (Safe for 0 or "").
