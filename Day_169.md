# Learn gRPC

- Engine `@grpc/grpc-js`
- Translator `@grpc/proto-loader`
- Parsing `protoLoader.loadSync(...)`
  - **`keepCase: true`**: Keep field names exactly as they are written in the `.proto` file.
  - **`longs: String`**: Represent it as a safe JavaScript text `String`
  - **`enums: String`**: Represent it as a safe JavaScript text `String`
  - **`defaults: true`**: If a client sends an object but forgets to include a field
  - **`oneofs: true`**: property can be Type A or Type B, but not both
- `loadPackageDefinition(...)`: (raw translated object) To (interactive JavaScript object tree)
- `server.addService`: links your **contract blueprint** to your **actual logic**
- `server.bindAsync(..)`
  - `127.0.0.1:50051`: The socket address
    - **`127.0.0.1`**: `localhost` (this local machine).
    - **`50051`**: Standard default port convention used for gRPC traffic
  - `grpc.ServerCredentials.createInsecure()`: Defines the network security wrapper (Just for local development)
  - `bindAsync`: locks the server onto that port and starts listening
