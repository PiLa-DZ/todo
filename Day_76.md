# Node.js

- Global Keyword: Node.js vs. Browser
- Package Management: Global vs. Local Installation
  - On Linux Don't use sudo npm! It messes up file ownership. Instead
  - Change NPM's default directory to a folder in your $HOME directory.
- NPX: The Node Package Executor
- Semantic Versioning (SemVer)
  - The Structure: `MAJOR.MINOR.PATCH`
  - The NPM Wildcards (Range Specifiers)
    - The Caret (`^`) - "Compatible with" (Default)
    - The Tilde (`~`) - "Bug fixes only"
    - The Absolute Version
